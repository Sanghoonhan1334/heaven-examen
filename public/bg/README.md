# 배경 이미지 폴더

천국고시 세계관을 위한 배경 이미지를 이 폴더에 추가하세요.

## 필요한 이미지

1. **hill.png** - 하단 언덕(동산) 배경
   - 가로로 넓은 곡선형 이미지
   - 브라우저 하단 15~25% 영역을 채움
   - 권장 크기: 1920x400px 이상

2. **cloud.png** - 중간 구름 배경
   - 카드들이 떠 있는 공간을 표현
   - 투명 배경 PNG 권장
   - 권장 크기: 1920x1080px 이상

3. **stairs.png** - 상단 계단 이미지
   - 천국으로 이어지는 계단
   - 아래에서 위로 이어지는 느낌
   - 권장 크기: 1200x200px 이상

4. **gate.png** - 상단 천국문 이미지
   - 목표와 이상향을 상징
   - Hero 타이틀 뒤쪽 또는 상단 중앙에 배치
   - 권장 크기: 400x400px 이상

## 이미지가 없을 때

이미지가 없어도 그라데이션 fallback이 자동으로 적용되어 사이트가 정상 작동합니다.
